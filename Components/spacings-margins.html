Spacing

margin

Utilities for controlling the outer spacing around an element.

Our system ships real margin utilities for all sides, both axes, single-side spacing, and negative offsets. The examples below focus on the most common spacing patterns already available in the framework.

Reference

These utilities are generated from the spacing token scale and can be applied directly in markup. The shipped steps are:

0, 2, 4, 6, 8, 12, 16, 20, 24, 32, 40, 48, 56, 64, 72

Class Styles
.m-8 margin: var(--spacing-8)
.mx-8 .my-8 Axis-based outer spacing utilities
.mt-6 .mr-4 .mb-8 .ml-2 Single-side margin utilities with shipped token values
.m-n8 .mt-n8 .mx-n8 Negative margin utilities

Basic example

Use .m-8 when the element needs even outer spacing on all four sides.

m-8
<div class="m-8">...</div>

Adding margin to one side

Use side-specific utilities like .mt-6, .mr-4, .mb-8, and .ml-2 to push an element away on one edge only.

mt-6
mr-4
mb-8
ml-2
<div class="mt-6">...</div><div class="mr-4">...</div><div class="mb-8">...</div><div class="ml-2">...</div>

Adding horizontal margin

Use .mx-8 when both left and right outer spacing should move together.

mx-8
<div class="mx-8">...</div>

Adding vertical margin

Use .my-8 when both top and bottom outer spacing should move together.

my-8
<div class="my-8">...</div>
