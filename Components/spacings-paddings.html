Spacing

padding

Utilities for controlling the inner spacing of an element.

Our system ships real padding utilities for all sides, horizontal and vertical axes, and single-side adjustments. Use the examples below to match the utility classes that already exist in the design system.

Reference

These utilities are already generated in the framework and map directly to the spacing token scale. The shipped steps are:

0, 2, 4, 6, 8, 12, 16, 20, 24, 32, 40, 48, 56, 64, 72

Class Styles
.p-8 padding: var(--spacing-8)
.px-8 padding-left: var(--spacing-8); padding-right: var(--spacing-8)
.py-8 padding-top: var(--spacing-8); padding-bottom: var(--spacing-8)
.pt-6 .pr-4 .pb-8 .pl-2 Single-side padding utilities with shipped token values

Basic example

Use .p-8 when the element needs even padding on all four sides.

p-8
<div class="p-8">...</div>

Adding padding to one side

Use side-specific utilities like .pt-6, .pr-4, .pb-8, and .pl-2 when the spacing needs to change on only one edge.

pt-6
pr-4
pb-8
pl-2
<div class="pt-6">...</div><div class="pr-4">...</div><div class="pb-8">...</div><div class="pl-2">...</div>

Adding horizontal padding

Use .px-8 when the left and right sides should expand together.

px-8
<div class="px-8">...</div>

Adding vertical padding

Use .py-8 when the top and bottom spacing should move together.

py-8
<div class="py-8">...</div>
