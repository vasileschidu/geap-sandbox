Sidebar

The sidebar serves as a persistent vertical navigation area, allowing users to access primary sections or features of an application or website. It helps organize complex interfaces by grouping related content and providing quick access to key destinations, improving overall navigation efficiency and user orientation.

Layouts

Use the sidebar as part of a full screen composition, not only as an isolated navigation list. We support two primary layout patterns: an integrated contained sidebar inside a bounded workspace, and a dashboard sidebar attached directly to the left edge of the application shell.

Layout patterns

  • Integrated layout
  • Dashboard layout

Integrated layout

Use the contained sidebar when the navigation belongs to a bounded module, settings workspace, or documentation area. The sidebar keeps rounded corners and lives inside the main content container rather than attaching to the viewport edge.

Contained Navigation

Rounded, in-container navigation for bounded workspaces and settings surfaces.
